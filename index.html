Github+Hexo 建个人blog

发表于 2018-10-04

本来只是想找个地方写一下spark的学习笔记，然后想起了很久之前没有建成功的github.io。于是就开始建博客了，至于spark的学习笔记，就放在下一篇吧。

系统是mac，懒得看英文doc，就用了hexo，然后发现hexo的主题挺多的，各种风格都有，而且也很好修改。接下来还是直接记录搭建的步骤吧，搭建的过程中忘记截图了，所以图片直接采用了网上的。

1.github建io仓库

有github的账号，在github上新建仓库

注意博客的名字一定要和github账号名相同！

建完之后可以clone到本地，然后可以建一个index.html看一下效果。详见Github pages：
https://pages.github.com/

2.安装hexo

安装hexo之前首先要有Node.js和Git。安装 Node.js 的最佳方式是使用 nvm。

在mac上可以用homebrew先安装了wget，然后用wget安装Node.js。

wget

wget -qO- https://raw.github.com/creationix/nvm/v0.33.11/install.sh | sh

安装完成后，重启终端并执行下列命令即可安装 Node.js。

nvm install stable

然后在本地新建一个文件夹hexo，在文件夹里安装hexo。

npm install -g hexo-cli

安装完成之后

即可访问网址： http://localhost:4000/

3.将hexo部署在git上

打开hexo文件夹下的_config.yml文件，在最下面，修改deploy为以下代码，repository后面的是1中新建的的仓库地址

然后再打开github.io就是hexo建的博客的样子了。

4.写博客

Hexo默认使用markdown语法来写作，可以用以下命令来创建一篇新文章：

hexo new [layout] <title>

其中，[layout]可以指定文章的布局，默认是post，hexo默认有三种布局，每种布局的路径不同：

新建完文章之后，就可以在hexo文件夹下相应的路径下找到新建的文章，然后使用markdown软件来写，写完保存之后，在终端里回到hexo的文件夹下，用下边的指令将内容更新到本地和io网站上。

首先 hexo g

然后 hexo s，可以在本地打开。

或者 hexo d，可以在io网站上打开。

5.更改主题

首先去hexo官网上寻找喜欢的主题，这里推荐next和yalia的主题。在github上clone到hexo的themes文件夹下。

然后完全可以按照next官网上的进行配置。
http://theme-next.iissnan.com/getting-started.html

配置完成之后在按照下边的指令即可完成主题的修改。

6.插入图片的Tips

这里将本地的图片插入文章中，有两种方式。

一种是直接将图片放在source下的images文件夹下，然后在文中引用的时候路径写成(/images/xx.jpg)就可以了。

另外一种是将把主页配置文件中的_config.yml里的post_asset_folder: 这个选项设置为true;

在hexo目录下执行npm install hexo-asset-image --save，这是下载安装一个可以上传本地图片的插件;

然后，运行hexo n “xxxx”来生成md博文时，/source/_posts文件夹内除了xxxx.md文件还有一个同名的文件夹;

最后在xxxx.md中想引入图片时，先把图片复制到xxxx这个文件夹中，然后只需要在xxxx.md中按照markdown的格式引入图片：![](图片.jpg) 。(注意：这里只写图片名即可，不用添加任何路径！)

以上基本上就是github+hexo搭建博客的内容了，更多的主题和内容的探索以后用到了再说吧。
